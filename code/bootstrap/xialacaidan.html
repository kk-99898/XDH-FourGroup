下拉菜单

.dropdown 类用来指定一个下拉菜单。

.dropdown-menu 类来设置实际下拉菜单。

我们可以使用一个按钮或链接来打开下拉菜单， 按钮或链接需要添加 .dropdown-toggle 和 data-toggle="dropdown" 属性。
