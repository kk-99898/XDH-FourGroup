对移动设备，确保适当的绘制和触屏缩

```
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

**viewport meta** 标签添加 *user-scalable=no* 可以禁用其缩放（zooming）功能

设置 *max-width* 为 100% 会重写任何通过 width 属性指定的宽度。这让图片对响应式布局的支持更友好。
